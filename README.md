# SciPy2026-entrepreneur-risk

This repo contains materials used to prep and present a 25-min talk for the 2026 SciPy conference.

The PPTX file was used in conjunction with the chart.html file, which contains a Plotly file with an interactive data visualization entitled "Entrepreneurship: Capital / Time / Failure Risk by Business Category."

Please see the separate markdown file discussing the data sources and varying levels of precision for different parameters.
